This query hunts for MITRE ATT&CK T1003 (OS Credential Dumping) across three Windows sub-techniques: T1003.001 (LSASS memory access via `OpenProcessApiCall` in `DeviceEvents` and known dumping tools like Mimikatz/comsvcs MiniDump), T1003.002 (SAM database extraction via `reg save`, volume shadow copy, or secretsdump-style commands in `DeviceProcessEvents`), and T1003.003 (NTDS.dit file access in `DeviceFileEvents` plus extraction commands via ntdsutil, diskshadow, or vssadmin). All five detection branches are unioned into a single result set showing `DeviceName`, `AccountName`, `ProcessName`, and `CommandLine` alongside a `DetectionType` label that maps each hit to its specific sub-technique for triage prioritization.
