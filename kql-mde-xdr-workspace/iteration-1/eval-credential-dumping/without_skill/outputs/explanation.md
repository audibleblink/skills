This query hunts for MITRE ATT&CK T1003 (OS Credential Dumping) across three primary Windows techniques: LSASS memory access (via tools like Mimikatz, ProcDump, and comsvcs MiniDump), SAM database extraction (via `reg save`, secretsdump, and similar utilities), and NTDS.dit access (via ntdsutil IFM creation, Volume Shadow Copy abuse, and direct file operations on `ntds.dit`). It unions results from both `DeviceProcessEvents` and `DeviceFileEvents` to surface process-level command lines alongside file-level NTDS.dit copy/access events. Each result row surfaces device name, account name, and the full command line to give analysts immediate triage context.
