# Basic Formatting

| Style              | Syntax                              |
| ------------------ | ----------------------------------- |
| **Bold**           | `**text**` or `__text__`            |
| *Italic*           | `*text*` or `_text_`               |
| ~~Strikethrough~~  | `~~text~~`                          |
| ==Highlight==      | `==text==`                          |
| ***Bold & italic***| `***text***`                        |

## Headings

```md
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## Lists

```md
- Unordered item
* Also unordered
+ Also unordered

1. Ordered item
2. Second item

- [x] Completed task
- [ ] Incomplete task
- [?] Custom status
- [-] Cancelled
```

## Horizontal Rule

```md
---
***
___
```

## Blockquotes

```md
> Quoted text
> Continues here
```

## Escaping

Backslash before special characters: `\*`, `\_`, `\#`, `` \` ``, `\|`, `\~`
