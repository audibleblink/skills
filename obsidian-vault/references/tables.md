# Tables

```md
| Left-aligned | Center-aligned | Right-aligned |
| :----------- | :------------: | ------------: |
| Content      | Content        | Content       |
```

Escape `|` inside table cells with `\|` (needed for wikilinks with aliases and image resizing):

```md
| Column 1 | Column 2 |
| -------- | -------- |
| [[Note\|Alias]] | ![[image.jpg\|200]] |
```
